# 건강운동관리사 문제 소스 확장 메모

## 현재 반영

- 공식 KSPO 원본 기준 A형 문제은행: 2015~2024년 반영
- 자동 배포 사용 가능 문항: 1,485문항
- 그림/표 의존 문항: PDF에서 crop 가능한 항목은 `assets/question-images/`에 이미지 연결
- 이미지가 없는 `<그림>`/`<표>` 문항은 자동 배포에서 제외

## 2025년 처리 상태

- 공식 KSPO 2025년 문제지/최종정답 원본은 `materials/raw/kspo/2025/`에 확보
- 문제지 PDF가 텍스트 PDF가 아닌 스캔 이미지라 일반 `pdftotext` 추출 불가
- 로컬 OCR 환경은 구성 완료: `tesseract-lang` 설치, `kor+eng` 인식 가능
- 단, OCR 결과가 보기 번호와 영문/숫자 혼합 부분에서 오인식이 있어 자동 배포 투입 전 별도 검수 필요

## 운영 원칙

- 1순위: 공식 KSPO 기출 원문 + 최종정답
- 2순위: 기출 해설은 링크/개념 보강 참고만 사용
- 3순위: 외부 예상문제는 출처·해설·이용권한이 확인된 경우에만 별도 `external-*` bank로 분리
- 자동 배포에는 `verified=true`, `wrongRateBasis`가 있는 문항만 사용
